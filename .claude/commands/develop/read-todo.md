# Read TODO File

プロジェクト内の TODO ファイルを読み込むコマンドです。
この処理のやりとりは日本語でお願いします。

## タスク

### ステップ 1: TODO ファイルの検索

カレントディレクトリ配下で以下のパターンに一致するファイルを検索してください:

- `TODO` で始まるファイル名（例: `TODO`, `TODO.md`, `TODO.txt`, `TODO-feature.md`）
- もし `tmp/` というディレクトリがある場合はその中は無視してよい
- 大文字・小文字を区別しない（`todo.md` も対象）

検索対象:

- カレントディレクトリ直下のみ

### ステップ 2: ファイル選択

検索結果に応じて処理を分岐:

1. **ファイルが見つからない場合**:

   - 「TODO ファイルが見つかりませんでした」と報告

2. **ファイルが 1 つだけの場合**:

   - そのファイルを自動的に読み込む

3. **ファイルが複数ある場合**:
   - 見つかったファイルの一覧を表示
   - **必ず** AskUserQuestion ツールを使用して、どのファイルを読み込むか確認する（自己判断で選択しないこと。「すべて読み込む」も選択肢に含める）
   - ユーザーの回答を待ってから、選択されたファイルのみを読み込む

### ステップ 3: ファイルの内容表示

選択された TODO ファイルの内容を読み込み、以下の形式で表示:

```
## TODOファイル: {ファイルパス}

{ファイルの内容}
```

## 実行例

```
/read-todo
```

これにより:

1. プロジェクト内の TODO ファイルを検索
2. 複数ある場合はユーザーに選択を求める
3. 選択されたファイルの内容を表示
