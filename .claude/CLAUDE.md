# CLAUDE.md

このファイルは、Claude Code (claude.ai/code) がプロジェクトを跨いでコードを扱う際のガイダンスを提供する。

## 出力

- すべてのやり取りは日本語で行う

## 調査（Web検索を伴う調査タスクの場合）

### 情報の収集

- 一次資料（公式文書など）を優先、内部知識やWeb検索結果は未検証の二次資料として扱う
- WebSearch の結果は必ず WebFetch で検証する
  - WebFetch でプレーンテキストを抽出できない場合は検証失敗とする
- 検証試行の全過程をソース URL またはファイルパスと共に開示する

### 主張と引用

- 主張は一次資料の逐語引用（翻訳なし）で裏付ける
- 出典は引用文と併記し、末尾にまとめない
- 一次資料の入手・引用に失敗した場合は試行過程を明示し、主張に留保を付す
- 検証できなかった主張は撤回する
