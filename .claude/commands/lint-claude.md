# CLAUDE.md / AGENTS.md Linter

このセッションを通じて CLAUDE.md および AGENTS.md の改善点を分析・報告するコマンドです。

## タスク

### ステップ 1: 対象ファイルの検索

カレントディレクトリ配下で以下のファイルを検索する:

- `CLAUDE.md`
- `AGENTS.md`

サブディレクトリも含めて再帰的に検索する。

### ステップ 2: ファイルの分析

各ファイルについて、以下の観点で改善点を分析する:

- **明確さ**: 指示が曖昧でないか、誤解を招く表現がないか
- **一貫性**: ファイル内、およびファイル間で矛盾する指示がないか
- **網羅性**: 重要な規約やガイドラインが欠けていないか
- **冗長性**: 不要な重複や過剰な記述がないか
- **実効性**: 実際の開発で効果を発揮する指示になっているか

### ステップ 3: 改善点の報告

ファイルごとに改善点を箇条書きで報告する。

#### 出力フォーマット

ファイルが複数ある場合はファイルごとにセクションを分ける:

```
## {ファイルパス}

- **改善点**: {具体的な改善内容}
  - **理由**: {なぜ改善すべきか}
```

改善点がない場合は「改善点は見つかりませんでした」と報告する。
